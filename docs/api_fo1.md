## `getFO1SaveProp`

**Params**

* [`FO1SAVE*`](#fo1save) `save`
* [`FO1SAVE_PROPS`](#fo1save_props) `prop`
* `void**` `destination`

**Returns**

`bool`

**Examples**

```c

bool result = getFO1SaveProp(save, FO1SAVE_PROPS_ENGINE_VERSION, (void**)&engineVersion);

    printf("Property: %u\n", engineVersion);
}
```
