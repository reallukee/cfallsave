# Fallout 2 Save File Format

> [!NOTE]
> Fallout 2 Format IS VERY CLOSE to Fallout 1 Save Format

| Property       | Type           | CFAllSave Type |
| :------------- | :------------- | :------------- |
| Save Signature | char[18]       | char[18]       |
| Player Name    | char[32]       | char[32]       |
| Save Name      | char[32]       | char[32]       |
